# Zapcart Admin

This is the admin application for the Zapcart e-commerce platform.

## Features
- **Products Management**: View and manage product listings.
- **Shared UI/Lib**: Uses monorepo shared packages for consistency.

## Tech Stack
- **Next.js 16** (App Router)
- **Tailwind CSS v4**
- **Lucide React Icons**
- **Inter Font**

## Development
```bash
npx turbo dev --filter=admin
```
